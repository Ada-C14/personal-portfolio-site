At Ada
Ruby Calculator My first Calculator, which can calculate "(a+b) / (c%d)" when you give "(a+b)" as one element and "(c%d)" as another element. Ex (3-5)*(4%2)=0, (4)**(2)=16, or (6)+(5/2)=8.5.

Ruby OO ride share Applied OOP and TDD to build the code to be able to load lists of passengers, trips and drivers from CSV files. Moreover, when a passenger requests a trip, we will pick a driver and make a new trip for the passenger.
