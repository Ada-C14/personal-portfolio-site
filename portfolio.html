icon commission calculator.

i sometimes do art commissions where i draw emotes for people's tabletop rpg characters. this is a calculator i coded to figure out how much certain commissions would cost.

my response to the coding challenge i submitted in my ada application. i'd like to think i've come a long way from here!

calculator.

my very first project as a cohort 14 adie student. yes, it's another calculator. but it also helped me conquer a weird fear of regex.

solar system.

a CLI program written at ada to learn about some of our planets in the solar system. currently only supports two planets, but i'd say they're pretty neat ones. you can even add info about your own!

grocery store.

library code written for an ada project to store orders of a grocery store using object-oriented programming. it was one of the first projects where we'd be writing our own program tests.

nightowl.

an ongoing owlbot for the #nightowl channel in the ada cohort 14 slack. one day it might just be a ruby on rails app. #hoot
