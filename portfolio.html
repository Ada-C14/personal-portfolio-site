Adagrams

An anagram is a word or phrase ftsormed by rearranging the letters of a different word or phrase, it has a specific scoring system, so that the player's submitted anagram scores poin. While working on Adagrams, it helped to think of a physical metaphor for the game, such as other common word games like Scrabble, these analog games all feature a pool of letter tiles that the player draws from. In this version of Adagrams, we only worked with the English alphabet.

Rideshare

We built a Ruby library that loads lists of passengers, trips and drivers from CSV files, provided methods for exploring this data, and provided a way to request a new trip. We used our understanding of classes, methods and attributes to create an object-oriented ride share system.

SlackCLI

For this project we used the Slack API in order to let us list users and channels, get details on users and channels, and send messages to users or channels, all from a Ruby program. Why we interacted with Slack using its API, and not its web-app? There are many places where you might wanted to use Slack, where it doesn't make sense to have a human-friendly web-app in the loop.
