Pair-program Slack CLI is a Ruby interface using the external dependency Slack API to provides a client with a workspace's channel and user data. The CLI can send messages to a desired channel or user.

The OO Rideshare can import CSV rideshare data and help a client create a new rideshare trip using the imported driver data.

Grocery Store is a TDD exercise, creating class instances and methods out of CSV grocery store data.

Solar System command line interface utilizes Ruby classes to allow a user to view/modify/create solar system data.
