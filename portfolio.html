Book Review Site

Practice writing HTML and CSS by building a static book review site.

Main goals:

  • Demonstrate an understanding of HTML syntax
  • Reach the understanding of appropriate tag selection for specific content
  • Show an ability to link local pages and remote pages
  • Practice with the creation of a external stylesheet
  • Organize a site with semantic HTML

Solar System

Create a CLI program that interacts with one Solar System and many Planets using OO principles.

Learning goals:

  • Create a class to tie together data and methods
  • Create a class which manages a collection of other objects
  • Write code that interacts with classes
  • The initialize stores parameters as instance variables with appropriate reader methods

Slack-CLI

Practice working with APIs by writing a command line application that interacts with the Slack API. Pair project. Wrote a complete command-line application that interacts with Slack. This application is able to List users and channels, see details about a user or channel, send a message to a user or channel.

Main goals:

  • Sending requests to and receiving responses from an API
  • Working with data from an external source
  • Mocking an external dependency during testing
  • Designing and implementing a program using inheritance
